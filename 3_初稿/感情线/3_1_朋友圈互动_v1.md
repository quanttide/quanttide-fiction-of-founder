# 朋友圈互动

傍晚。
她刷到了他的朋友圈：“我压力好大！我好想谈恋爱！”

稀稀拉拉的几个评论五花八门的。
“谈了恋爱压力更大。”
“我知道，我就这么一说。”
“给你介绍对象。”
“哈哈哈谢谢你，我就这么一说。”
每个评论都回复了。还有一句统一回复：
“哈哈哈谢谢大家关心。我这么喊就是找个理由让大家乐一下。之前有好友说，看我这么喊很好玩。”

没有一个人问他怎么了。
想到这里，她的心头突然一酸。
手控制不住地打字，“你怎么啦？”
回过神来，消息已经发出去一小会了。
再删除也不好，他会看到删除记录。

那就这样吧。她心一横。
很快收到了回复，“TAT 只有你关心我！感激不尽！代码进度又卡了 5555”
又好笑又心酸。她戳进去私信框拍了拍他。

过了几分钟，他也拍了拍她。
“拍一拍没有消息提醒，我都没有看到。”
“嘿嘿好开心٩(๑^o^๑)۶”
“只有你会关心我呢。”
她的心头酸酸胀胀的，有很多想说的话说不出口。
“我在你常去的便利店，要不要下来坐会？”她问。
“好哇！马上就来马上就来！”
