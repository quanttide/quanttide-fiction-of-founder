# 便利店谈心

他坐在24小时便利店靠窗的位置，看着雨丝从路灯的光线中穿过。

他感受着身体里堆积了一整周的疲惫，在手机备忘录上敲着脑子里流淌的思绪，有时候按着语音输入键说几句。

自动开关的玻璃门叮咚叮咚地一阵阵响，不断地有人进进出出。

都不是她。

他看了一眼最近一条她说出门过来的消息，只是十分钟之前。

他又切回到备忘录软件，从工作文件夹切换到生活文件夹，写道，“我还是太心急了。再等等吧。”


她急匆匆地下楼，到楼下看到下雨，又急匆匆地上楼拿伞，又急匆匆地下楼。

她越走越快，也顾不上风吹着雨打湿左边的衣服。

希望他别等急了，答应他五分钟就到的。

再走几分钟，终于前面亮起的便利店招牌。

她在门口合上潮湿的伞，等着门开了以后就冲了进去。

他抬起头。这次终于是她了。他喜笑颜开地朝她挥了挥手。

她走过去，把伞递给他，又走到收银台买关东煮。他一边收着伞，一边看她点菜，“要萝卜、香菇、豆腐泡、鱼丸、虾丸…对，两份都要一样的。”

她挨着他坐下，把一份放在他面前，另一份放在自己面前。

“我请你啦，别和我客气。”她看到他要开口，急忙说道。

“好。”他戳了一个鱼丸，“其实我很少吃关东煮。”

“你的豆浆喝完了？”

“喝完了，杯子扔垃圾桶了。我差点就准备回去了，今天运气真不错。”

“好饿，让我先把关东煮炫完。”她风卷残云地扫掉了半碗。看着他慢条斯理地戳着，还有点心不在焉地看着窗外，想了想没开口。

“我刚才在想爸爸妈妈。”他突然开口，“书上说，父母在，不远游。我却好像一直在一步一步走远一点。不过这也是他们希望的。”

他翻出来一张父母的合照给她看。

“他们感情很好。”她看了照片后，点了点头。

“大家都这么说，都说一看照片就能看出来。”

“我经常坐在这里想他们。有时候觉得很有压力，有时候又觉得很愧疚。”

她看他眼眶似乎有点微润。

“我会给家里寄点他们爱吃的东西，如果觉得自己照顾不上他们的时候。”

“我也想，但是我爸妈总说这个也不要那个也不要。搞得我也不知道怎么办才好。”

“那就不问。我也从来都不问的。”

“也是。”他挠了挠头。“他们怕我麻烦，我也是真嫌麻烦。”

“哈哈哈哈哈o(*≧▽≦)ツ”

他看着她有点嘲笑的眼神，假装生气地表现出一副无奈的表情。

窗外不知道什么时候雨停了。
