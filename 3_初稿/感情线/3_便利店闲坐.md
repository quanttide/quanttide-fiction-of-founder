# 便利店谈心

他坐在24小时便利店靠窗的位置，看着雨丝从路灯的光线中穿过。

他感受着身体里堆积了一整周的疲惫，在手机备忘录上敲着脑子里流淌的思绪，有时候按着语音输入键说几句。

自动开关的玻璃门叮咚叮咚地一阵阵响，不断地有人进进出出。

都不是她。

他看了一眼最近一条她说出门过来的消息，只是十分钟之前。

他又切回到备忘录软件，从工作文件夹切换到生活文件夹，写道，“我还是太心急了。再等等吧。”


她急匆匆地下楼，到楼下看到下雨，又急匆匆地上楼拿伞，又急匆匆地下楼。

她越走越快，也顾不上风吹着雨打湿左边的衣服。

希望他别等急了，答应他五分钟就到的。

再走几分钟，终于前面亮起的便利店招牌。

她在门口合上潮湿的伞，等着门开了以后就冲了进去。

他抬起头。这次终于是她了。他喜笑颜开地朝她挥了挥手。

她走过去，把伞递给他，又走到收银台买关东煮。他一边收着伞，一边看她点菜，“要萝卜、香菇、豆腐泡、鱼丸、虾丸…对，两份都要一样的。”

她挨着他坐下，把一份放在他面前，另一份放在自己面前。

“我请你啦，别和我客气。”她看到他要开口，急忙说道。

“好。”他戳了一个鱼丸，“其实我很少吃关东煮。”

“你的豆浆喝完了？”

“喝完了，杯子扔垃圾桶了。我差点就准备回去了，今天运气真不错。”

“好饿，让我先把关东煮炫完。”她风卷残云地扫掉了半碗。看着他慢条斯理地戳着，还有点心不在焉地看着窗外，想了想没开口。

“我刚才在想爸爸妈妈。”他突然开口，“书上说，父母在，不远游。我却好像一直在一步一步走远一点。不过这也是他们希望的。”

他翻出来一张父母的合照给她看。

“他们感情很好。”她看了照片后，点了点头。

“大家都这么说，都说一看照片就能看出来。”

“我经常坐在这里想他们。有时候觉得很有压力，有时候又觉得很愧疚。”

她看他眼眶似乎有点微润。

“我会给家里寄点他们爱吃的东西，如果觉得自己照顾不上他们的时候。”

“我也想，但是我爸妈总说这个也不要那个也不要。搞得我也不知道怎么办才好。”

“那就不问。我也从来都不问的。”

“也是。”他挠了挠头。“他们怕我麻烦，我也是真嫌麻烦。”

“哈哈哈哈哈o(*≧▽≦)ツ”

他看着她有点嘲笑的眼神，假装生气地表现出一副无奈的表情。

“又有点馋上次的麻辣烫了。”她突然安静下来。

“还是在想白天的事。”他轻轻问。

“嗯。”她的声音有点闷闷的。

“今天你见的那个创业者确实有点离谱。”他吐槽道，“在一个企业软件里放一大堆排名榜，正经功能不好好做，想着什么游戏化，也不知道是被哪路神仙忽悠了。”

被戳中心事的她心里暖暖地化开了一些。

“还是你的嘴毒，而且一阵见血。”

“那必须的。”

“自从自己开始当老大以后，就感觉破事越来越多，一点点心事都会把自己压垮。”她苦笑道，“在当老大这方面，还是干了十年的你最有经验了。”

“一开始都难。当了十年以后越来越难。欲望永远都在膨胀。有时候我也在反思，是不是欲望过于旺盛了。”

“但是好像大家很喜欢被我的欲望激励的样子，感觉干活格外有劲。”

她听着他在耳边的声音，望着窗外的路灯，想到了十年前第一次看到他的公开演讲的样子。

那时候的他是个意气风发的少年郎，站在讲台上向着全校介绍自己的科研进展。

十年了。现在的他成熟稳重地像一滩清水，感觉什么事情都好像波澜不惊一样。

他这十年来过的肯定不容易。真的不知道一个活泼开朗的人是怎么变成现在这样沉默寡言的。

她本想说“第一次看你演讲的时候…”话到嘴边又停住了。她赶忙换了个话题。

“如果创业者都能像你这样就好了。感觉这个市场和我想的一点都不一样。”

“哎，是啊。最后还是我们这些做事的人背锅。光是和客户解释，AI 不是许愿瓶是复读机，就要商务经理们多费多少脑细胞。”

“你带的真好。他们都很能打。”

“你也可以的，相信我。你也是一个很好的领导。”

说着“领导”，他突然想起了初中政治老师在班里经常提到她老婆时候的称呼是“我家最高领导”，眼角弯成了月牙。

“你又想到什么开心的事情啦？”她有点好奇地试探。

“我今天劝新入职的技术助理不要太卷。半个小时前还在问我问题，我给他劝去睡觉了。”他赶忙找了个理由，害怕一不小心说漏嘴了。

“这样你就可以继续偷懒了。”她吐槽着想，心里想的肯定是别的事情，表情都不一样。

“没错，员工快乐工作了，老板才能快乐偷懒。不然一天天的要被他们卷的没有生活了。”

“哈哈哈哈哈哈哈哈哈哈(*≧▽≦)”

他也跟着她一起笑了起来。

窗外的雨渐渐停了。
