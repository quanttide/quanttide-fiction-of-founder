# 便利店谈心

他坐在24小时便利店靠窗的位置，看着雨丝从路灯的光线中穿过。

他感受着身体里堆积了一整周的疲惫，在手机备忘录上敲着脑子里流淌的思绪，有时候按着语音输入键说几句。

自动开关的玻璃门叮咚叮咚地一阵阵响，不断地有人进进出出。

都不是她。

他看了一眼最近一条她说出门过来的消息，只是十分钟之前。

他又切回到备忘录软件，从工作文件夹切换到生活文件夹，写道，“我还是太心急了。再等等吧。”


她急匆匆地下楼，到楼下看到下雨，又急匆匆地上楼拿伞，又急匆匆地下楼。

她越走越快，也顾不上风吹着雨打湿左边的衣服。

希望他别等急了，答应他五分钟就到的。

再走几分钟，终于前面亮起的便利店招牌。

她在门口合上潮湿的伞，等着门开了以后就冲了进去。

他抬起头。这次终于是她了。他喜笑颜开地朝她挥了挥手。

她走过去，把伞递给他，又走到收银台买关东煮。他一边收着伞，一边看她点菜，“要萝卜、香菇、豆腐泡、鱼丸、虾丸…对，两份都要一样的。”

她挨着他坐下，把一份放在他面前，另一份放在自己面前。

“我请你啦，别和我客气。”她看到他要开口，急忙说道。

“好。”他戳了一个鱼丸，“其实我很少吃关东煮。”

“你的豆浆喝完了？”

“喝完了，杯子扔垃圾桶了。我差点就准备回去了，今天运气真不错。”

“好饿，让我先把关东煮炫完。”她风卷残云地扫掉了半碗。看着他慢条斯理地戳着，还有点心不在焉地看着窗外，想了想没开口。

“……我其实不太确定自己在做什么。”她先开口了，声音比刚才轻。“当初想做孵化器，是因为读到过一些东西。有人把想象中的世界写得那么好，好到我觉得，那个世界是可以达到的。我想帮那样的人。想离那个世界近一点。”

她顿了顿。“但真的做起来，每天都在处理琐事。合同、场地、BP、团队纠纷。有时候一整周都想不起当初那个世界长什么样。”

她抬起头看他。“你会这样吗。”

他望着碗里的热气。

“我做的产品，最近被人说不够快。不够高效。他们想要更多提醒、更多排行榜、更多让人停不下来的东西。”

他停了一下。“但我不想做那个。我想做的是让人能停下来的东西。看清楚自己到底要什么。和谁一起。”

“很难卖。”

她没说话。她把自己碗里的竹轮夹起来，放进他碗里。那是她刚才一直没动过的。

“这个也好吃。”她说。

“你尝尝。”

他低头看着那块竹轮。表面微微炸过，浸了汤之后变得柔软。

他咬了一口。

“怎么样。”

“……好吃。”

她笑了一下，低头继续吃自己碗里的。

过了一会儿，她说：“所以今天算逗号。”

他夹起那块萝卜。很烫。萝卜的甜和汤的咸混在一起。豆浆还在桌上，彻底凉了。但她碗里的热气还在往上升。

窗外不知道什么时候雨停了。
