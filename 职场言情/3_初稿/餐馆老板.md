# 餐馆老板

他认真地听着老板的滔滔不绝，“我们最怕遇到客人说分量少。打工的客人吃的多饭量大，不像那些游手好闲的。以前我们出去打工，吃的也多。现在在这上班，吃的少多了。”

他们一起走在回去的路上。

她说，“听到老板说这算上班的时候我有点恍惚。原来打工和上班在不同的人眼里是那么不同。”

他说，“我很喜欢和店主们聊天。他们的倾诉欲好多都很强，能从他们身上听到很多故事。”

她说，“如果能把这些故事都记录下来就好了。很希望他们被看到。”

他说，“我很喜欢他们。我觉得比精英圈子里那群道貌岸然的精英有意思地多。”

她说，“现在的你，身上一股土洋混合的感觉。一边说着泥土里一样的大白话，一边又说着横跨 N 个学科没人跟得上的概念。给人一种莫名其妙的混搭感。”

“我觉得我和你在一起的时候，很容易回到这种不土不洋的原生态。和你在一起很放松。”他躲开她的视线看向远方。

她的心里被戳了一下。

“我喜欢观察这些餐馆的变化。看着一些餐馆倒下，另一些餐馆取而代之，看着市场的变迁，感知它的脉络。无数次被他们安抚焦躁的内心。”

所以，他是经历了多少焦躁不安。她想了想，没问出口。

“大隐隐于市。这种感觉还蛮好的。”
