# 提纲

这是一部校园恋爱小说。男女主是以厦大为原型的鹭岛大学的大二学生。男主窦逸是经院的学术天才，女主林栀是新传的院花。

## 情节

相遇：他们在医院偶然相遇，两个人初见就互生好感。女主主动拦下男主，男主让男主带他上楼。男主一开始犹豫，又因为看到女主摔倒而折返。女主没有留联系方式，而是选择说了一句“有缘再见”。
相识：事后男主回味并且在网上发帖说这句话让他开心了一整天。他们在医院的照片被第三方拍到，事情引起了轰动。男女主因此重新联系。
相知：官方出了一份对于论坛热点事件的分析报告。根据当事人回应的论坛不成文惯例，男女主在校园访客中心的火锅店约饭讨论回应贴，因此深入了解彼此公开和私下的样子。回到宿舍以后，他们各自珍藏了论坛的新照片，并且和自己的好友们回味这次约会。

女主：林栀
生日2005年1月，摩羯座，大二。
小名只只/栀栀。
姓名
栀子花的花语是“喜悦、坚强、永恒的爱与约定，一生守候我们的爱”。
她的父母或许相识于栀子花盛开的夏季。父亲第一次送花给母亲，便是一束芬芳的栀子。于是，“林栀”便成了他们爱情最芬芳的纪念，寓意她是“爱情的结晶”。
家庭
- 场景：那或许是一个大学校园的初夏傍晚，刚下过一场小雨。父亲在路边的小花园里，鼓起勇气折下一支带雨的栀子花，递到母亲手中。花香混着雨后泥土的清新，瞬间俘获了母亲的心。
- 信物：那束花可能不名贵，甚至只是从老师家院子里“借”来的几朵。但母亲却将它小心翼翼地压在厚重的书本里，做成了干花书签，至今仍珍藏在她最心爱的那本诗集里。
- 家庭传统：于是，家里便有了一条不成文的规定——每一年，在父母相遇纪念日那天，也就是栀子花即将盛开的时节，父亲都会为母亲和林栀各送上一束栀子花。
- 氛围熏陶：林栀从小就在这清甜的香气中长大。她熟悉这花香，就像熟悉妈妈温柔的怀抱和爸爸宽厚的肩膀。她从小就懂得，爱是具象的，它可以是夏日里的一阵芬芳，是父亲归家时手中不变的花束，是父母相视一笑时眼里的星光。
