# 创作历程

根据我们之前的对话，以下是你告诉我的、关于你创作心理过程的信息梳理：

1. 这两对男女主，最早源于同一个版本的最初设定：他们可以通过某种时空穿越机制进行交流或见面。后来，你觉得这样写“太科幻”“太网文”“有点写不好”，就放弃了这个设定。
2. 于是，这个最初的构思，演变和拆分成了现在的两个独立故事。
3. 校园言情这篇是先写的，它的主角也源于最早的版本。故事的起点，是你本人在那个年纪经历过的一次遗憾：一次医院里的相遇，一个心动的开局，但当时没有勇气开始。这篇小说的创作动机，就是去探索一个假设：如果当时开始了，会怎么样？
4. 职场言情这篇，来源于两个部分的合并：
   · 一部分，是从校园言情小说里拆分出来的“男二女二”。他们的年龄设定在28-32岁左右，职业是创业者和创业孵化器负责人。你最初的构想是让他们因为男女主的事件而偶遇。
   · 另一部分，是你写的另一部小说，主角年龄设定在25-26岁，核心是“双向暗恋”的剧情。
5. 你把上述这两个部分融合在了一起，最终形成了现在的职场言情故事。融合的结果，是让那个“双向暗恋”的情节得以更充分地展开。
6. 在写职场言情这第二本小说时，你探索的是另一个更深层的假设：如果当时他们没有开始，但却在各自的世界里默默关注着对方，那么十年之后，他们重新相遇，会怎么样？
